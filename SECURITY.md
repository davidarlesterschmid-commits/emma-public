# Security

This repository is a **transparency / audit snapshot**. It does not host runnable application code or deployment configuration from the emma private monorepo.

**Reporting security issues**

- If you have a **security-relevant** finding: please **do not** post exploit details in public issues.
- Use **GitHub private vulnerability reporting** for this repository if it is enabled by the owner, or contact the repository owner through the **official channel** they publish for the emma product (when available).

**Scope**

- Reports about **this snapshot repository** (e.g. broken links, accidental disclosure in audit text) are welcome.
- Vulnerabilities in the **unpublished** application or backend are outside what this tree can address; they should be directed to the owner’s private security process when offered.

**No warranty**

- Audit documents are point-in-time; they are not a substitute for a penetration test, formal compliance sign-off, or product security program.
