# emma_app

Dieses Verzeichnis ist das Ziel fuer die Endkunden-App im kuenftigen Monorepo.

Die minimalen Dart-/Flutter-Dateien sind bereits angelegt, damit VS Code,
Packages und Importgrenzen vorbereitet sind. Vor echten Device- oder Web-Starts
sollten die Plattformordner in diesem Verzeichnis vervollstaendigt werden,
typischerweise mit `flutter create .` innerhalb von `apps/emma_app`.
